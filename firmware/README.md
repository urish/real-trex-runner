# T-Rex Game Firmware for Raspberry Pi Pico

## Pinout

| GPIO | Peripheral    | Pin  |
| ---- | ------------- | ---- |
| 2    | Status LED    |      |
| 3    | WS2812 LED    | DIN  |
| 4    | DFPlayer      | RX   |
| 5    | E-Paper       | CS   |
| 6    | E-Paper       | CLK  |
| 7    | E-Paper       | DIN  |
| 8    | E-Paper       | DC   |
| 9    | E-Paper       | RST  |
| 10   | E-Paper       | BUSY |
| 11   | KY-024 Sensor | DO   |
| 12   | Rail Motors   | nSLP |
| 13   | Rail Motors   | DIR  |
| 14   | Rail Motors   | STEP |
| 15   | Jump Motor    | nSLP |
| 16   | Jump Motor    | DIR  |
| 17   | Jump Motor    | STEP |
| 18   | Button        | IN   |
